## File Structure

- `main.py`: Main Python script containing the code for the dashboard.

- `park_info.py` and `kpi_tracker.py`: Secondary Python scripts containing the code for the dashboard's different pages.

- `attendance.csv`, `entity_schedule.csv`, `link_attraction_park.csv`, `waiting_times.csv`: CSV files containing the data used by the dashboard.

## CSV File Requirements

- All CSV files used by the dashboard should be placed in the same directory as the dashboard script (`dashboard.py`).
- Ensure that the CSV files contain the required data fields and are properly formatted for use with the dashboard.
